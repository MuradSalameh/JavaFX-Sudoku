Sudoku Regeln

Die leeren Felder im 9x9 Raster so ausfüllen, dass

- in jeder Zeile
- jeder Spalte
- jedem 3x3 Block

jede Zahl nur 1 Mal vorkommt.


Algo:

 - zum Start prüfen ob bedinungen eingehalten werden - ansonsten abbrucht des Spiels mit Fehlermeldung
 - Wenn bedingungen eingehalten, dann Spieler lösung suchen lassen, falls es eine gibt / 
   Alternativ button um Lösung von Algo suchen lassen, falls es eine gibt
   

Sudoku anlegen:
9x9 Grid
9x9 Array
Leerstellen = 0;

					wenn nicht gefunden dann return false.
					- jedes element in zeile, spalte block prüfen ob gefunden[] wahr ist.
